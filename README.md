# Afnaan Ahmed P — Personal Portfolio

This is the source code for my personal portfolio website, featuring a unique, cinematic, and dark-themed "Upside Down" aesthetic heavily inspired by Stranger Things and modern tech design. 

The portfolio is designed to showcase my skills as an M.Tech Software Engineering student specializing in full-stack development, machine learning, cloud architecture, and system design.

## Live Website

🔗 **[https://22MIS1157.github.io](https://22MIS1157.github.io)**

You can use this link to share the portfolio with recruiters, include it on your resume, or send it to friends.

## Features

- **Cinematic Experience**: A custom preloader and Stranger Things inspired intro sequence.
- **Scroll-Driven Parallax**: Advanced JavaScript `requestAnimationFrame` mechanics to handle scroll-based typography and layout transitions smoothly without lag.
- **Interactive UI**: Custom cursor replacements, magnetic buttons, and dual-color cyberpunk hover glitch effects.
- **Responsive Layout**: fully optimized for both desktop and mobile devices.
- **Pure Web Technologies**: Built using semantic HTML5, modern vanilla CSS3, and JavaScript—without relying on heavy frameworks, ensuring blazing-fast load times.

## Local Development

To run this project locally:
1. Clone the repository: `git clone https://github.com/22MIS1157/22MIS1157.github.io.git`
2. Navigate into the directory: `cd 22MIS1157.github.io`
3. Serve the directory using any local web server. For example, using Python:
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and navigate to `http://localhost:8000`
